# 🎮 Jogando Ping-Pong 🎮
### Este é um projeto simples do famoso jogo Pong implementado em Python usando a biblioteca Pygame.

![Fig.gif](img/parte.gif)

# Requisitos

### - Python 3.x 🐍
### - Pygame 🎮

# Instalação dos Pacotes 🔧

```bash
pip install -r requirements.txt
```

# Executando o jogo ▶️

```bash
python main.py
```
